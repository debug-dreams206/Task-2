\# Internship Task 2 – Website Project



\## Project Overview

This project is developed as part of \*\*Internship Task-2\*\*.  

The goal of this task is to demonstrate the use of a \*\*templating engine (Nunjucks)\*\* with a \*\*modern development bundler (Vite)\*\* to create a clean, reusable, and visually appealing website.



---



\## Technologies Used

\- \*\*Nunjucks\*\* – for reusable templates and layouts  

\- \*\*Vite\*\* – for fast development server and bundling  

\- \*\*HTML, CSS, JavaScript\*\* – structure, styling, and interactivity
