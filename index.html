INTERNSHIP TASK 2

Project Overview

This website is developed as part of Internship Task-2. The goal is to demonstrate the use of a templating engine with a modern development bundler.

Technologies Used

  • Nunjucks – for reusable templates and layouts
  • Vite – fast development server and bundler
  • HTML & CSS – structure and styling

Features Implemented

  • Reusable Header & Footer
  • Base layout with template inheritance
  • Clean folder structure
  • Local development server

Learning Outcomes

Through this task, I learned how to structure a web project, create reusable UI components, and serve compiled HTML using a modern toolchain.
